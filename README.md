# SafePass

SafePass — это мобильное приложение для управления паролями, позволяющее хранить, генерировать, копировать и просматривать пароли с помощью биометрической аутентификации.

## Особенности

- Сохранение и хранение паролей для разных сервисов
- Генерация сложных паролей
- Копирование пароля и логина в буфер обмена
- Детальный просмотр информации о пароле в модальном окне
- Биометрическая аутентификация (FaceID, TouchID) для защиты доступа к паролям
- Поддержка темной и светлой темы интерфейса
- Многоязычность (русский и английский языки)

## Технологии и язык программирования

- **Язык:** JavaScript
- **Фреймворк:** React Native
- **Библиотеки:**
  - `expo-local-authentication` — биометрическая аутентификация
  - `expo-clipboard` — работа с буфером обмена
  - `@react-native-async-storage/async-storage` — локальное хранение паролей
  - `expo-status-bar` — управление статус баром
  - React Native стандартные компоненты и хуки (`useState`, `useEffect` и др.)

## Основные компоненты

- **App.js:** главный контейнер приложения, управление состоянием, темой и модалками
- **PasswordList.js:** список паролей с биометрической защитой
- **AddPasswordModal.js:** модальное окно добавления пароля
- **PasswordDetailsModal.js:** модальное окно с деталями и возможностью копирования
- **utils/passwordGenerator.js:** генерация случайных паролей
- **utils/storageHandler.js:** работа с локальным хранилищем
